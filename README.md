# 🔐 Brute Force Simulator PRO (Educational)

A cybersecurity educational tool that demonstrates how brute force attacks work using mathematical simulation.

## ⚠️ Disclaimer
This project is for **educational purposes only**.
It does NOT attack real systems.

## Features
- Hacker-style terminal UI
- Password strength analysis
- Estimated crack time
- Animated progress bar
- Speed meter
